# E-Commerce Website with Admin Dashboard

## Overview
A modern e-commerce web application built with Next.js and Tailwind CSS. The project includes a responsive shopping interface and an admin dashboard UI for managing products and monitoring store activities. Users can browse products, add items to cart, and interact with different front-end features designed to simulate a real-world online shopping experience.

This project focuses mainly on front-end development, component structuring, responsive design, and state management.

---

## Features
- Responsive e-commerce user interface
- Product listing section
- Add to cart functionality
- Shopping cart UI updates
- Product detail pages
- Admin dashboard interface
- Responsive navigation and layout
- Modern UI design with Tailwind CSS
- Reusable components
- Dynamic routing with Next.js

## Development Process
The project was built using a component-based architecture with Next.js to improve scalability and code organization. Tailwind CSS was used for styling and responsive layouts, allowing faster UI development and consistent design across pages.

The application structure was divided into reusable components such as product cards, navigation bars, cart sections, and dashboard panels to improve maintainability and reduce repetitive code.

---

## Challenges & Solutions

### Managing Cart State
One of the challenges was handling cart updates across different components while keeping the UI synchronized. This was solved by organizing state properly and ensuring components updated dynamically whenever products were added to the cart.

### Responsive Dashboard Layout
Creating a dashboard layout that worked smoothly across desktop and mobile devices required careful use of Tailwind CSS grid and flex utilities. The layout was adjusted to maintain usability on smaller screens.

### Component Reusability
Structuring reusable components without making the codebase difficult to manage was another challenge. This was improved by separating UI sections into smaller reusable components with cleaner props handling.

## Installation & Setup

Clone the repository:
```bash
git clone https://github.com/preciousnaza/next-js-ecommerce.git
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
